# Eagle OppaiOracle Tagger

Eagle プラグイン: 選択した画像に対して、ONNX ベースの画像タグ付けモデル OppaiOracle でローカル推論を行い、タグを自動付与します。

## 機能概要

- 選択画像への一括タグ付け（Window プラグイン）
- OppaiOracle V1.1 ONNX モデル（448x448）による推論
- グローバル閾値スライダー（0.0〜1.0）
- 最大タグ数設定（1〜100）
- タグマージ戦略切替（追加 / 上書き / 差分）
- 進捗表示・キャンセル対応
- 設定の自動保存（localStorage）
- NSFW タグ注意警告（初回のみ）
- ブラックリスト機能

## 要件

- Eagle 4.0 Build12 以降
- Windows x64（確認済み）
- macOS / Linux は未検証
- 初回起動時にモデルダウンロード（約 1GB）のためのインターネット接続

## インストール

1. 配布 zip ファイルをダウンロードする
2. zip を展開する
3. 展開したフォルダを Eagle のプラグインフォルダに配置する（メニュー > プラグイン > プラグインフォルダを開く）
4. Eagle を再起動する
5. メニュー > プラグイン > OppaiOracle Tagger を起動する
6. 初回起動時にモデルをダウンロードする（約 1GB、5 分程度）

## 使い方

1. Eagle 内でタグ付けしたい画像を選択する
2. メニュー > プラグイン > OppaiOracle Tagger を起動する
3. 初回のみモデルダウンロード画面が表示されるので「ダウンロード」をクリックする
4. 閾値・最大タグ数・マージ戦略を設定する
5. 「実行」ボタンをクリックする
6. 進捗バーが完了するまで待つ（キャンセル可能）
7. 完了サマリを確認する

## 設定項目

| 項目 | 説明 | デフォルト |
|------|------|-----------|
| 閾値 | タグとして採用する確率の下限（0.0〜1.0、0.05 刻み） | 0.5 |
| 最大タグ数 | 1画像あたりの最大タグ数（1〜100） | 30 |
| マージ戦略 | 既存タグとの結合方法（追加 / 上書き / 差分） | 追加 |
| ブラックリスト | 除外するタグ名（カンマ区切り） | 空 |

## 注意事項

- 本プラグインは OppaiOracle モデルを使用します。このモデルは汎用アニメ画像タガーであり、Danbooru 系データセットに由来する NSFW タグを語彙に含みます。結果の運用は自己責任で行ってください。
- 初回起動時に約 1GB のモデルファイルをダウンロードします。ダウンロードは中断後もレジューム可能です。
- 推論はすべてローカル環境で実行されます。画像データが外部に送信されることはありません。
- AI による自動推論のため、出力されるタグには誤りが含まれる可能性があります。必ず人手で確認することを推奨します。

## トラブルシューティング

**モデルのダウンロードに失敗する**
- インターネット接続を確認する
- ダウンロードを再試行する（レジューム対応）
- モデルファイルを手動で HuggingFace（Grio43/OppaiOracle）からダウンロードし、`models/V1.1/` に配置する

**プラグインがプラグイン一覧に表示されない**
- Eagle 4.0 Build12 以降であることを確認する
- プラグインフォルダのパスが正しいか確認する
- Eagle を再起動する

**推論が遅い**
- 1 画像あたり 5 秒以内が目安です（CPU 性能に依存）
- 大量画像を一度に処理する場合は時間がかかる可能性があります
- タスクマネージャーでメモリ使用量を確認する
